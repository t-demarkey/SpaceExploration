# Space Exploration App 

A cross-platform Flutter application that brings space science to life through interactive learning and real-time astronomical data.

## Overview
The Space Exploration App is built with Flutter and Firebase, combining educational experiences with live space data. Users can explore astronomy topics, take daily quizzes, track planetary visibility based on location, and view NASA’s Astronomy Picture of the Day — all within a smooth, modern UI.

## Features
- **Daily Space Quizzes** with a dynamic, real-time leaderboard  
- **Learn Tab** featuring rotating astronomy topics  
- **NASA Astronomy Picture of the Day** integration  
- **Location-Based Planet Visibility** tracking  
- **Live Countdown** to the next rocket launch  
- **Video Lessons** powered by the YouTube Player API  
- **Firebase Authentication & Firestore** for secure, real-time data handling

## Tech Stack
- **Frontend:** Flutter (Dart)
- **Backend:** Firebase Auth, Firestore, Cloud Functions  
- **APIs & Services:** NASA APOD API, YouTube Player API, Geolocation Services
